# 🎯 GIT

**Sistema de Control de Versiones Distribuido**

## ¿Qué es?
- Control de versiones distribuido
- Cada clone = repositorio completo
- Trabajo offline posible
- Creado por Linus Torvalds (2005)

## Conceptos Core
- **Working Directory**: Archivos actuales
- **Staging Area**: Cambios preparados
- **Repository**: Historial completo
- **HEAD**: Dónde estás ahora

## Filosofía
- Piensa en snapshots, no en diffs
- Las ramas son punteros ligeros
- Todo es local primero
- Sincronización explícita con remoto
