# Cloud Computing

> [!abstract] Objetivo
> Entender los conceptos más importantes y saber cuales son las ventajas y desventajas de utilizar el cloud computing

---

## 📝 Definición

El cloud computing es el uso de servicios de computación a través de internet

---

## ⚙️ Conceptos Clave

### 📌 Ventajas

**Tipo**: Beneficios

**Características**:
- Reducción de Costos
- Rapidez y flexibilidad
- Escalabilidad elástica
- Seguridad y fiabilidad
- Innovación

**Relacionado**: [[Modelos de Servicios de Cloud]]

---

### 📌 Desventajas

**Tipo**: Riesgos y restricciones

**Características**:
- Dependencia de internet
- Dependencia del proveedor
- Lock-in y centralización
- Almacenamiento de datos
- Escalabilidad a largo plazo

**Relacionado**: [[Seguridad en la nube]]

---

### 📌 Modelo de Responsabilidad Compartida

**Tipo**: Marco de seguridad

**Características**:
- Responsabilidad del proveedor (infraestructura física, red, hipervisor)
- Responsabilidad del usuario (datos, aplicaciones, accesos)
- Responsabilidad Variable según modelo (IaaS vs PaaS vs SaaS)

**Relacionado**: [[IaaS vs PaaS vs SaaS]]

---

### 📌 Mercado del Cloud Computing

**Tipo**: Tendencias

**Características**:
- Crecimiento potencial
- Distribución geográfica
- Sectores impulsores
- Enfoque en eficiencia

**Relacionado**: [[Tendencias Tecnológicas 2025]]

---

## 💻 Ejemplo Práctico

### Caso Netflix
Netflix usa AWS (Amazon Web Services) para:
- **Escalabilidad**: Millones de usuarios simultáneos
- **Distribución global**: Contenido cerca de cada usuario
- **Reducción de costos**: No mantiene datacenters propios
- **Innovación**: Experimenta con nuevos servicios rápidamente

---

## 💭 Reflexiones & Conexiones

Para la tecnología de IA/ML hoy en día es fundamental el cloud computing para que modelos que exigen mucha capacidad computacional funcionen para cualquier usuario que tenga conexión a internet, sin necesidad de tener un data center físico personal.
