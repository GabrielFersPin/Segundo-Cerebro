# Amazon Textract
Is fully managed service that can detect and extract text and data from sacanned documents, PDFs, and images. One of the use cases for Amazon Textract is to process invoices and receipts. For example, Amazon Textract can detect billing and shipping addresses automatically from images.

# Amazon Transcribe
Is a service taht you can use to convert speech into text. You can use Amazon Transcribe to facilitate the transcription of audio recordings. If media contains domain-specific or non-standard terms, you can use a custom vocabulary or a custom model to improve the accuracy of the transcriptions.

# Amazon Rekognition
Rekognition is a deep learning image and video analysis service. You can use Amazon Rekognition to analyse and extract insights from visual content. One of the use cases is the classification of products into categories by using custom labels and trainning a model. To meet the requirements, you must provide labeled images of trainning.
