## 📌 Resumen

Un **Bosque Aleatorio (Random Forest)** es un método de **aprendizaje de conjunto (ensemble learning)** muy potente y popular, que se utiliza tanto para tareas de **clasificación** como de **regresión**. Funciona construyendo una multitud de árboles de decisión durante el entrenamiento y, para la predicción, toma la moda de las clases (en clasificación) o la media de las predicciones (en regresión) de los árboles individuales.
