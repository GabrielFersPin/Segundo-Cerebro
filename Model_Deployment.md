# Model Deployment

Model deployment is a integration of a machine learning model into a production environment where it can be used to make predictions on new data.

### Self-Hosted API

In self-hosted API you host your own ML models on your own infrastructure. Using virtual machines or containers. This approach involves more control over the model and the data, but requires more technical expertise.

### Managed API

Managed API services are cloud-based services that provide a fully managed environment for deploying and hosting your ML models as APIs.SageMaker is a example of this service.
