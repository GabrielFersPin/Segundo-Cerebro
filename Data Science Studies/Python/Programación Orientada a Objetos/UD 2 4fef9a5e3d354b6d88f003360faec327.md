# UD 2

- Encapsulamiento
    
    Nos permite agrupar datos y controlar su comportamiento en nuestra clase
    
- Herencia
    
    Nos permite generar una jerarquía de clases en las que podemos compartir funcionamientos comunes y en el que existirá una clase padre o superclase y una o varias clases hijas (subclases).
    

## Encapsulamiento en la programación orientada a objetos

- Nomenclatura
    
    Para indicar que es privado y solo puede ser utilizado dentro de la clase
    
    ```python
    def __init__(self, value):
    	self._value = value
    ```
    
- try
    
    Se ejecuta normalmente hasta que llega al except
    
- except
    
    Se busca la excepción que podría ocurrir
    
    - ZeroDivisionError
        
        excepción predefinida en Python que se genera cuando se intenta dividir un número por cero 
        

## Control de acceso a los atributos de una clase: getter y setter

- getter
    
    Un método que se utiliza para leer el valor e un atributo de un objeto. Se puede aplicar cualquier validación antes de devolver el valor
    
- setter
    
    Establecer o modificar el valor de un atributo de un objeto. Se puede aplicar cualquier validación 
    

## Decoradores

- @property
    - Permite modificar el comportamiento de un atributo, convirtiendo ese atributo en una propiedad
    - solo lectura
- @decorador
    
    nombre de una clase o método que toma otra función o clase como entrada y devuelve una nueva función o clase
    
- @atributo_setter
    
    Crea métodos que permitan modificar un atributo privado (setter) de manera controlada.
    

## Descriptores

- Definición
    
    Un descriptor es un objeto, y por lo tanto pertenece a una clase, que se utiliza para acceder a un atributo o propiedad de un objeto de manera especial 
    
- **get**
    
    **get** para acceder al atributo
    
- **set**
    
    para establecer el valor del atributo
    

## Herencia

- Superclase
    
    Clase padre, la que crea la clase
    
- Subclases
    
    Las clases que heredan los atributos y métodos de la clase padre
    
- Super
    
    Llama a un método de la clase base que haya declarado el constructor
