# AWS Service Tools for Respo

AWS responsible AI service tools help to ensure that machine learning models are developed, deployed, and used in a responsible and ethical manner.

## Foundation model evaluation

**Model evaluation on Amazon Bedrock**

With Amazon Bedrock you can evaluate, compare, and select foundation models to use in your applications.

- Automated evaluation
- Custom evaluation
- Human evaluation

**Model evaluation on Amazon SageMaker AI Clarify**

Supports FM evaluation. You can automatically evaluate the output of your FM for bias and fairness

## Safeguards for Gen AI applications

**Amazon Bedrock Guardrails**

You can implement guardrails for your Gen AI applications based on your use cases and responsible AI policies.

### Bias Detection

**Amazon SageMaker AI Clarify**

Supports bias detection. You can automatically evaluate the output of your FM for bias and fairness

**Amazon Sagemaker Data Wrangler**

Helps you preprocess data and detect data bias.

### Model prediction explanation

**Amazon SageMaker Clarify**

Provide scores datiling which features contributes the most to your model predictions.
